# Statecher

Statecher is a simple state machine engine for Spring Boot.

The idea is straightforward: instead of hardcoding workflow logic all over your services, define state transitions in configuration and let the engine handle the flow.

This helps keep business logic cleaner and makes workflows easier to change without digging through code.
